# ZEE5-Automation
This project automates browser testing for the [ZEE5](https://www.zee5.com) website using **Selenium** and **Python**.  
It demonstrates how to use automation to test key website features like navigation, search, and result validation.
# FEATURES:-
Launches the ZEE5 website automatically  
 Handles cookie pop-ups  
 Clicks on the search icon  
 Enters a movie or show name  
 Validates that search results are displayed  
 Takes screenshots for verification  
 Closes the browser automatically after execution  
# CONCEPT USED
-Selenium WebDriver  
- XPath and CSS selectors  
- Browser automation  
- Exception handling  
- Screenshot capture  
- Python functions and modular structure
# TOOLS USED
Python - Main programming language 
Selenium- Web automation framework 
ChromeDriver - Controls Chrome browser 
VS Code- Development IDE
